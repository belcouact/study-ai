About this website

This is a small experimental project fueled by curiosity, exploration, and passion. It began with several lingering questions:

  1. With generative AI being so powerful, how can we adapt it to personalized needs?
  2. In the AI era, can programming and creation truly become accessible to everyone?
  3. How can we leverage AI to assist children's learning?

Here are some insights gained during implementation:

  • Reject complacency, start thinking
  • Resist procrastination, take immediate action
  • Embrace trial and error, grow through practice
  • Never give up, keep exploring
  • Believe in yourself, success will follow

Tools used in the project (all learned during implementation):

  • Content provider: DeepSeek
  • Coding: Claude-3.7-sonnet
  • Code Editor: Cursor / VS Code
  • Web deployment: Github / Cloudflare
